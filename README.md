# Spam Email Classifier

## Project Overview
This project is a Machine Learning-based Spam Email Classifier that detects whether a message is Spam or Ham (Not Spam).

## Technologies Used
- Python
- Pandas
- Scikit-Learn
- Google Colab
- Naive Bayes Algorithm

## Dataset
SMS Spam Collection Dataset

## Features
- Detects Spam Messages
- Detects Normal Messages
- Fast and Accurate Classification
- Machine Learning Based Prediction

## Model Used
Multinomial Naive Bayes

## Accuracy
99.19%
